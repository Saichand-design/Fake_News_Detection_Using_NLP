# Fake_News_Detection_Using_NLP
This project uses Natural Language Processing techniques to classify news articles as real or fake.

**Key Highlights**
Text cleaning & preprocessing,
Stopword removal & stemming,
TF-IDF vectorization,
Built classification models:
Logistic Regression,
Naive Bayes,
Passive Aggressive Classifier,
Evaluated using F1-score & confusion matrix,
Model deployment ready.

**NLP Techniques Used**
Tokenization
Lemmatization
TF-IDF
N-grams

**Tech Stack**
Python | NLTK | Scikit learn | Pandas | Streamlit
